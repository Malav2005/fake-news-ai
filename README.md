# 📰 Explainable Fake News Detection using Machine Learning and Generative AI

## 📌 Overview
This project is a Machine Learning-based Fake News Detection system that classifies news as real or fake. It also provides simple explanations to make the results understandable and trustworthy.

---

## 🎯 Features
- 🧠 Fake news detection using Machine Learning  
- 🔍 Explainable AI output  
- 🌐 REST API using Flask  
- ⚡ Real-time prediction system  

---

## 🛠 Tech Stack
- Python  
- Flask  
- Scikit-learn  
- Pandas
